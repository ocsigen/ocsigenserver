Start the server with some instructions. Never returns. It takes as main parameter a list of virtual hosts (see Ocsigen_server.host below).

Options behave exactly like their configuration file counterparts.

type instruction = 
  Ocsigen_extensions.virtual_hosts ->
  Ocsigen_extensions.config_info ->
  Ocsigen_lib.Url.path -> Ocsigen_extensions.extension

The type of instructions to be used inside an host or site. Instructions are defined by extensions (Staticmod, Eliom, etc.)
